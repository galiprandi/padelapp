## Priority Issues

### [P0] Skeleton structure mismatch in page.tsx
ProfileFormSkeleton shows old avatar selector with 3 toggle buttons, but actual form has static display only. Fix: update skeleton to match new structure.

### [P1] Auto-save status visibility
"Guardando…" is small, bottom-placed. No indicator that changes are pending before 800ms debounce. Fix: add visual indicator near input showing pending state.

### [P1] Empty state for initials fallback
When displayName is null, getInitials returns "?" which feels broken. Fix: use User icon or better default.

### [P2] Security link card clarity
Card doesn't clearly indicate it's navigation. Fix: add chevron-right icon.

### [P3] Mobile touch target for "Usar mi foto de Google"
Text link is too small for mobile touch. Fix: convert to proper button with adequate touch target.
